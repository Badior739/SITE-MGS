# Prompt pour Créer des Onglets comme Services et Équipe

## Objectif
Créer un composant d'onglets réutilisable qui peut afficher différents contenus (Services, Équipe, etc.) avec une navigation par onglets élégante et interactive.

## Structure Technique

### 1. Composant Principal : TabsComponent
```typescript
interface TabItem {
  id: string;
  label: string;
  icon: LucideIcon;
  content: React.ComponentType;
  badge?: string | number;
}

interface TabsProps {
  tabs: TabItem[];
  defaultTab?: string;
  className?: string;
  variant?: 'default' | 'pills' | 'underline' | 'cards';
}
```

### 2. Fonctionnalités Attendues

#### Navigation par Onglets
- **Onglets horizontaux** avec icônes et labels
- **Indicateur actif** animé avec Framer Motion
- **Badge de notification** optionnel sur les onglets
- **Responsive design** (dropdown sur mobile)

#### Animations et Interactions
- **Transition fluide** entre les contenus
- **Effet hover** sur les onglets
- **Animation de l'indicateur** de sélection
- **Swipe gestures** pour mobile

#### Variants de Design
- **Default**: Onglets classiques avec underline
- **Pills**: Onglets en forme de pilules
- **Cards**: Onglets style cartes
- **Underline**: Simple ligne en dessous

### 3. Intégration avec les Composants Existantants

### 4. Implémentation Suggérée

#### Structure des Fichiers
```
components/
├── ui/
│   ├── Tabs.tsx (composant principal)
│   ├── TabItem.tsx (onglet individuel)
│   └── TabContent.tsx (contenu)
├── pages/
│   ├── ServicesPage.tsx (page avec onglets)
│   └── TeamPage.tsx (page avec onglets)
```

#### Utilisation Exemple
```typescript
const tabs = [
  {
    id: 'services',
    label: 'Services',
    icon: Briefcase,
    content: ServicesTab,
    badge: '8'
  },
  {
    id: 'team',
    label: 'Équipe',
    icon: Users,
    content: TeamTab
  },
  {
    id: 'portfolio',
    label: 'Portfolio',
    icon: Image,
    content: PortfolioTab
  }
];

<Tabs 
  tabs={tabs}
  defaultTab="services"
  variant="pills"
  className="w-full"
/>
```

### 8. Exemples d'Utilisation

#### Page Services Complète
```typescript
const ServicesPage = () => {
  const serviceTabs = [
    { id: 'web', label: 'Web', content: WebServices },
    { id: 'design', label: 'Design', content: DesignServices },
    { id: 'marketing', label: 'Marketing', content: MarketingServices }
  ];

  return (
    <div className="min-h-screen">
      <Hero />
      <Tabs tabs={serviceTabs} variant="cards" />
      <CTA />
    </div>
  );
};
```
